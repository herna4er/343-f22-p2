This app takes in movie names, or keywords of a movie's name, and displays a results of their reviews/rating from the nytimes movie reviews api and the omdb api.

Satisfied requirements:

  • deploy site: My site has been deployed to my github account
  • is responsive: My site has multiple media queries to handle the display based on the display size
  • use 2 different apis: Site uses a OMDB api to search up information of movies and uses the nytime movie reviews api to search up brief reviews of a movie
  • has search: The site has a search functionality to be able to look up movies/movies with similar names
  • results are filterable: Results can be filtered by their ratings and if they are critics picks or not
